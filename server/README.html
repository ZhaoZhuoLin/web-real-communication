使用Node解决Web实时通信

使用场景梳理

在日常开发中，整理出来关于：接入上下游数据通信的流程

  • 接收上游通过UDP下传的实时数据，并通过socket服务端与web端实时通信

  • web端使用socket的实时通信创建

  • socket下发分为：admin,user两个级别

  • admin可以查看总览数据

  • user查看业务数据

实现

创建WebSocket

// 创建一个express应用
var app = express();

// 创建一个 HTTP 服务器，Express 应用将使用该服务器
const server = http.createServer(app);

// 创建 WebSocket 服务器并使用 HTTP 服务器
const webSocketServer = new WebSocketManager(app);
webSocketServer.start();
